# Stitch Vibe Design Prompt

## 直接貼進 Stitch 的 Prompt

---

設計一個活動報名網站，主題是一間房地產集團的新總部開幕參觀活動。

### 氛圍
我希望使用者打開這個頁面時感覺到「被邀請參加一場隆重但溫暖的開幕派對」。整體風格是慶典感 + 專業感的平衡，不是那種廉價的促銷頁面，而是像收到一張有質感的邀請函。

### 背景與視覺效果
- 背景是柔和的淺灰色漸層
- 背景上有很多品牌色的氣球緩緩往上飄（深藍灰 #3f5262 佔 50%、珊瑚紅 #dd6650 佔 30%、明黃 #f2cf00 佔 20%）
- 氣球有景深效果：近的大而清晰、遠的小而模糊
- 整個頁面覆蓋一層淡淡的毛玻璃效果，讓氣球看起來有朦朧感
- 金色小紙花持續從天飄落，帶有微微發光

### 卡片風格
所有內容區塊都使用 Glassmorphism（毛玻璃）風格：
- 半透明白色底 + backdrop-filter blur + 白色細邊框
- 大圓角（24px）
- 輕微的陰影讓卡片浮起來

### 配色
- 主色：深藍灰 #3f5262（按鈕、標題）
- 強調色：珊瑚紅 #dd6650（重要標記、hover 效果）
- 點綴色：明黃 #f2cf00（慶祝元素）
- 按鈕 hover 時從深藍灰變成珊瑚紅

### 字體
Noto Sans TC，標題用 Black (900) 字重，內文用 Regular (400)

### 頁面結構（由上到下）

**1. 導航列**
- 固定頂部，滾動後變成毛玻璃效果
- 左邊：品牌字「WUOHOME」
- 右邊：膠囊型「立即報名」按鈕

**2. Hero 區塊（兩欄）**
- 左欄：
  - 小標籤 pill「2026.04.26（日）三重」帶脈動圓點
  - 大標題「窩的家集團 新總部開幕」
  - 副標題「誠摯邀請您蒞臨參觀」（珊瑚紅色）
  - 說明文字
  - 倒數計時器：4 個毛玻璃小卡片顯示 天/時/分/秒
  - 膠囊型 CTA 按鈕
- 右欄：
  - 毛玻璃卡片，列出活動資訊（日期、地點含地圖連結、免費參加）

**3. 跑馬燈**
- 全寬無限滾動，展示集團旗下 5 間公司名稱
- 每間公司用不同顏色區分

**4. 樓層介紹（5 張卡片堆疊）**
- 每張毛玻璃卡片代表一個樓層（1F-5F）
- 左側大字樓層號（半透明）
- 右側：公司名稱 + 彩色 badge + 說明 + 服務標籤 pills
- 1F 住商不動產、2F 室內設計、3F 租賃仲介、4F 社會住宅、5F 錄播室

**5. 報名區塊（兩欄）**
- 左欄：3 張場次選擇卡片（上午場/下午場一/下午場二）
  - 每張顯示時間、對象、名額進度條、狀態 badge
  - 點選後出現勾勾
- 右欄：毛玻璃表單卡片
  - 欄位：姓名、電話、單位、身份分類（下拉）
  - 大按鈕「確認報名」

**6. 溫馨提醒**
- 毛玻璃卡片，說明不收花籃
- 兩張贊助選項卡片（$2,000 / $3,000）
- 銀行匯款資訊表格

**7. Footer**
- 品牌字 + Slogan + Copyright

### 互動效果
- 滾動時元素淡入上移（scroll reveal）
- 卡片 hover 時微微上浮
- 按鈕 hover 從深藍灰變珊瑚紅
- 報名成功彈出 modal + 撒彩色紙花

### 參考風格
類似 Apple 產品頁面的毛玻璃質感，但更加溫暖有慶祝感。不要太冷調、不要太商務。

---

## 使用方式

1. 開啟 stitch.withgoogle.com
2. 建立新專案
3. 先匯入同資料夾的 `DESIGN.md`（讓 Stitch 讀取設計規則）
4. 再把上面的 Prompt 貼進去，開始生成
5. 生成後可以用語音跟 Stitch 微調：「把按鈕再大一點」「背景氣球再多一些」
